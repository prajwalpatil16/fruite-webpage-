## 📌 What This Project Does

A colourful and modern landing page for a fresh fruit and grocery brand. Focuses on visual appeal, clean layout, and responsive design — the kind of page you'd build for a small business or food delivery brand.

---

## ✨ Features

- **Eye-catching Hero Section** — bold product imagery and clear value proposition
- **Product Showcase** — featured fruit and grocery items with pricing
- **About Section** — brand story and freshness promise
- **Responsive Design** — fully mobile-optimised layout
- **Smooth Animations** — CSS transitions and scroll effects
